# Personal Finance Management App

A comprehensive personal finance management application built with Kotlin, Jetpack Compose, and Firebase.

## Tech Stack
- **Language:** Kotlin
- **UI:** Jetpack Compose + Material 3
- **Architecture:** MVVM + Clean Architecture
- **Local Database:** Room Database
- **Cloud Sync:** Firebase (Firestore, Auth, Storage, FCM)
- **Charts:** Vico
- **Biometrics:** AndroidX Biometric library
- **Export:** Apache POI, iTextPDF
- **Currency API:** ExchangeRate-API
